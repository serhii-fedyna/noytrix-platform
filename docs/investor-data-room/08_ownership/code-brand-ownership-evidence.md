# Code and Brand Ownership Evidence

Confidential IP diligence checklist.

## Code Ownership

Evidence to collect:

- GitHub repository ownership screenshot for `noytrix-platform`.
- Commit history showing founder authorship.
- Local repository history export if needed.
- List of nested product repositories and their remotes.
- Contributor list.
- Contractor agreements and IP assignments, if any.
- Open-source dependency list and license review.

## Brand Ownership

Evidence to collect:

- Noytrix name creation records.
- Logo source files and design ownership.
- Domain registration records.
- App store listing ownership.
- Google Play developer account proof.
- Social accounts and email accounts used for the brand.
- Trademark search results.
- Trademark application, if filed.

## Product Account Ownership

Accounts to document:

- GitHub.
- Google Play Console.
- Google Cloud.
- Expo/EAS.
- OneSignal.
- RevenueCat.
- OpenAI.
- Domain registrar.
- Hosting provider.
- Email provider.
- Telegram bot ownership.

## Chain of Title

Investor diligence should be able to answer:

1. Who wrote the code?
2. Who owns the code?
3. Were any contractors involved?
4. Did every contributor assign IP?
5. Who owns the brand?
6. Who controls the production accounts?
7. Can the company operate without personal accounts?

## Required Cleanup Before Sharing

Do not include:

- `.env` files.
- Service account JSON.
- Keystores.
- API keys.
- Raw databases.
- User personal data.
- Private SSH keys.
