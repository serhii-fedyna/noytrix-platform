## External Sharing Rule

Before sharing externally:

1. Replace all placeholders with verified facts.
2. Remove secrets, private keys, database exports, raw user data, and credentials.
3. Convert selected documents to PDF.
4. Add access control, watermarking, and a viewer log.
5. Have counsel review legal and securities-language sections.
