# Product Roadmap

Confidential operating roadmap.

## 0-30 Days

- Clean release process for mobile.
- Stabilize Google Play subscription flow.
- Complete Ukrainian translation coverage.
- Complete push language tagging after app update.
- Confirm analytics accuracy and remove test data from traction.
- Clean local repository artifacts before external diligence.
- Add production monitoring checklist.

## 31-90 Days

- Chrome extension production QA.
- Improve scan accuracy benchmark.
- Reduce false positives on legitimate websites.
- Add structured product analytics dashboard exports.
- Package API plans and developer documentation.
- Build B2B lead list and outreach workflow.
- Modularize backend risk engine from monolithic `main.py`.

## 3-6 Months

- PostgreSQL migration as primary data layer.
- Dedicated worker system for collectors, calendar, push, and analytics.
- Security review and privacy policy update.
- Extension store launch.
- API customer pilots.
- Partner community pilots.
- Add evidence-based reports for B2B users.

## 6-12 Months

- Enterprise dashboard.
- Wallet and platform integrations.
- Advanced threat graph UI.
- Higher-volume data ingestion.
- Formal model evaluation and accuracy reports.
- Internationalization expansion.
- SOC-style internal operations workflow.

## Strategic Goal

Make Noytrix the plain-language crypto risk layer across app, browser, API, and community channels.
